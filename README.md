# Mabalacat City Water District Website

A React-based website for the local water district in Mabalacat City, Pampanga.

## Features
- **Home Page**: Shows water services, operational stats, and customer testimonials
- **About Page**: Company background, mission/vision, team members, and core values  
- **Contact Page**: Office info, contact form, and location details
- **Mobile Responsive**: Works on phones and tablets
- **Professional Design**: Clean layout with Bootstrap styling

## Tech Stack
- React 19 (JavaScript)
- React Router for navigation
- Bootstrap 5 + Bootswatch theme
- CSS3 for custom styling

## Getting Started
```bash
# Install dependencies
npm install

# Start development server
npm start
```

Open http://localhost:3000 to view the website.

## Project Structure
- `src/pages/` - Main page components (Home, About, Contact)
- `src/components/` - Reusable components (Navbar, Footer, etc.)
- `src/data/content.js` - Website content and data
- `src/styles/` - CSS stylesheets
- `public/` - Static assets and HTML template
